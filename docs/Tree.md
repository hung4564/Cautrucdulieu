# Tree - cây

## Tree - Cây

Cấu trúc dữ liệu cây là cấu trúc dữ liệu gồm tập hữu hạn các nút, giữa các nút có mối quan hệ phân cấp gọi là quan hệ "cha - con".
Nút đầu tiên gọi là nút gốc.

### Binary tree - cây nhị phân

Là một dạng của cây với đặc điểm:

- Mọi nút trên cây chỉ có tối đa 2 nhánh con
